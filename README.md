# Cruise-of-the-Global-Fleet
Built a cruise management web application enabling users to explore cruise details and manage booking data through a structured frontend-to-backend workflow

🚢 Cruise of the Global Fleet
📌 Project Overview
Cruise of the Global Fleet is a full-stack web application designed to manage and explore cruise booking information.
The system allows users to register, log in, and interact with cruise-related data through a structured frontend-to-backend workflow.
This project demonstrates practical implementation of web technologies integrated with a Java-based backend architecture.
🛠️ Technologies Used
🎨 Frontend
HTML
CSS
JavaScript
⚙️ Backend
Java
JSP (Java Server Pages)
Apache Tomcat Server
🗄️ Database
MySQL
🔐 Authentication System
The application includes:
User Registration
User Login System
Session-based authentication
Users can successfully register and log in to the system.

⚠️ Current Status
Frontend interface is fully structured.
User authentication (Login & Registration) is functional.
Backend architecture is implemented using JSP and Java.
Some backend functionalities are under improvement due to integration and server-side configuration issues.
This project is being continuously refined to enhance complete backend workflow and database operations.
